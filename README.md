## 快速开发

🚀 本地开发

```bash
# 安装依赖
npm install # 或则 yarn

# 启动服务
npm start  # 或则 yarn start
```

⌨️ 提交代码

```bash
npm run commit
```

📖 更新日志

```bash
npm run release
```

📦 生产部署

```bash
# 测试环境
npm run build:test

# 预发环境
npm run build:pre

# 正式环境
npm run build:prod

# 项目文档
npm run build:docs
```

## 目录结构

* 导航配置：`src/config/menu.js`
* 路由配置：`src/config/routes.js`
* 路由入口：`src/router.js`
* 通用组件：`src/components`
* 布局文件：`src/layouts`
* 页面文件：`src/pages`
* 入口文件：`src/main.js`
